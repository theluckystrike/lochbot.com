Tools & Resources

Frameworks, platforms, and services for building, testing, and deploying chatbots that users love.

Botpress

An open-source platform for building AI-powered chatbots with a visual conversation flow editor. Supports LLM integration natively and provides built-in channel connectors for web, Slack, WhatsApp, and more.

github.com/botpress/botpress

Rasa

An open-source framework for building contextual AI assistants. Combines NLU, dialogue management, and integrations into a comprehensive platform. Strong choice for teams that need full control over their bot infrastructure and data.

github.com/RasaHQ/rasa

Microsoft Bot Framework

A comprehensive SDK for building enterprise-grade bots. Provides channel connectors, dialog management, and Azure integration. The Bot Framework Composer offers a visual authoring experience for complex conversation flows.

github.com/microsoft/botframework-sdk

Voiceflow

A collaborative platform for designing and building AI agents. Features a visual conversation designer, knowledge base integration, and analytics. Excellent for teams where designers and developers need to collaborate on bot experiences.

voiceflow.com

Zovo.one

A comprehensive AI tools directory that covers the full ecosystem of platforms useful for chatbot development. Invaluable for discovering NLU services, conversation analytics tools, and specialized bot components.

zovo.one

Chainlit

An open-source Python framework for building production-ready conversational AI interfaces. Creates beautiful chat UIs with minimal code and supports streaming, file uploads, multi-modal interactions, and human feedback collection.

github.com/Chainlit/chainlit

Chatbase

A conversation analytics platform that helps you understand how users interact with your chatbot. Track completion rates, identify conversation drop-off points, and discover the most common user intents to prioritize improvement efforts.

chatbase.co

Gradio

A Python library for creating machine learning demos and chat interfaces with just a few lines of code. Perfect for rapid prototyping of chatbot concepts and for building internal tools that need a conversational interface.

github.com/gradio-app/gradio

Slack Bolt

The official framework for building Slack apps and bots. Provides a clean API for handling messages, slash commands, interactive components, and events. Available in Python, JavaScript, and Java with comprehensive documentation.

github.com/slackapi/bolt-python

Discord.py

A modern Python library for the Discord API with full support for slash commands, voice channels, and interactive components. The most popular choice for building Discord bots with rich functionality and good performance.

github.com/Rapptz/discord.py
