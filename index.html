Chatbot Response Tester

Define a system prompt, send messages, and see how a chatbot would respond. Score each turn for relevance, tone, and length.

Note: This tool uses rule-based heuristics to simulate chatbot responses. It analyzes your system prompt for tone and keywords, then generates responses based on common chatbot behavior patterns (greetings, clarifying questions, problem handling, closing). It is not powered by actual AI — it demonstrates how structured prompts influence response style.

Common System Prompt Mistakes

Too vague

"Be helpful" lacks specifics. Try: Respond in 2-3 sentences. Always ask one clarifying question.

No boundaries

Without limits, bots ramble. Add: If you cannot help, say "Let me connect you with a specialist."

Conflicting instructions

"Be concise" + "Explain in detail" confuses behavior. Pick one tone and stay consistent.

Missing persona

Without a role, responses lack personality. Start with: You are a [role] for [company].
